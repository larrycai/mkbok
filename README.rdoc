This application generate ebooks (.pdf,.epub,.mobi) from markdown plaintext files

== Install

Install:
    gem install mkbok

== Usage

Create new project:
    mkbok --create sample --title "learn mkbok"
    mkbok --build # needs pandoc,XeTex packages
    
For more help:
    mkbok --help
